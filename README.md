# Algoritmos y Estructuras de Datos en C

## Contenido
- **Algoritmos**
  - Búsquedas: Binaria, Lineal.
  - Ordenamiento: Burbuja, QuickSort, MergeSort, etc.
- **Estructuras de Datos**
  - Listas enlazadas (simples y dobles).
  - Árboles binarios (incluyendo AVL y otros).
